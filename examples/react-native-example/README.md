# React Native Demo

## Usging Expo-Cli

Install client
> npm install -g expo-cli

Update project dependencies
> npm install

Run expo @ browser
> expo start

Run expo @ ios
> expo ios

Run expo @ android
> expo android

## Old School

Start react-native packager
> react-native start

Run @ ios
> react-native run-ios

Run @ android
> react-native run-android
